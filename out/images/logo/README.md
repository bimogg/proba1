# Логотип для навбара

Загрузите сюда логотип Nexus для навбара.

## Имя файла:
`nexus-logo.png`

## Путь:
```
public/images/logo/nexus-logo.png
```

## Рекомендуемые размеры:
- Ширина: 40-60px
- Высота: 40-60px
- Формат: PNG (с прозрачным фоном) или SVG

## Альтернативные имена:
Если ваш файл называется по-другому, измените путь в `src/components/Navbar.tsx`:
```tsx
src="/images/logo/ваше-имя-файла.png"
```
