# Изображение для Hero секции

Загрузите сюда главное изображение для Hero секции (справа от текста).

## Имя файла:
`megaphone-illustration.png`

## Путь:
```
public/images/hero/megaphone-illustration.png
```

## Рекомендации:
- Формат: PNG (с прозрачным фоном) или SVG
- Размер: рекомендуется квадратное изображение (например, 600x600px или больше)
- Содержание: иллюстрация с мегафоном и иконками вокруг

## Если файл называется по-другому:
Измените путь в `src/components/Hero.tsx` на строке 56:
```tsx
src="/images/hero/ваше-имя-файла.png"
```
