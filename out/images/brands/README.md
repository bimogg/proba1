# Логотипы брендов

Загрузите сюда логотипы компаний для секции Brands.

## Имена файлов:
- `amazon.png`
- `dribbble.png`
- `hubspot.png`
- `notion.png`
- `netflix.png`
- `zoom.png`

## Путь:
```
public/images/brands/
```

## Рекомендации:
- Формат: PNG с прозрачным фоном или SVG
- Размер: примерно 120-150px по ширине
- Цвет: можно в градациях серого (grayscale), при наведении будет цветным

## Если файлы называются по-другому:
Измените имена в `src/components/Brands.tsx` в массиве `brands`.
